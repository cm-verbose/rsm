# PNG

There are two test suites included within this directory:

- `image-test-suite/` contains images from the [imagetestsuite](https://code.google.com/archive/p/imagetestsuite/downloads) Google SVN repository.

- `png-suite/` contains images from the [PngSuite](http://www.schaik.com/pngsuite) by Willem Van Schaik.
